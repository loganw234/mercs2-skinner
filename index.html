mercs2-skinner

paint new skins for existing Mercenaries 2 models UNVERIFIED
Nothing this tool produces has been loaded in the game yet. The DXT1/UCFX encoder is byte-identical to the community project's reference script, but no skin built here has been injected and seen on screen. The additive inject_parts --repoint path in step 3 is written from the tool's documented interface and has not been run end to end.

1Bundle

Everything comes from that folder — the decoded PNGs, the UV layout, the material bindings and the original block bytes the new asset is cloned from.

2Texture

Save the sheet with the wireframe on to get a paint template, edit it in any image editor, then bring it back with Replace.

3Export

A · Modkit mod recommended

Replaces the character's own textures. The modkit does the encode, the container, the WAD packing and the merge with your other mods — no command line. Best if you just want to recolour something.
